大組網頁同步方法

方法一:

1.黑窗先進入本地倉儲 ex:cd tmp/cd2022bg3

2.查看遠端倉儲路徑 git remote -v 如未設定過upstream只會跑出兩行

3.git remote add upstream https://github.com/409232xx/cd2022bgx.git把409232xx的cd2022bgx倉儲設置為自己的upstream，執行此指令後無返回任何訊息，再次執行指令 git remote -v 檢查是否成功。

4.git status 檢查本地已沒有未提交之修改

5.git fetch upstream 抓取409232xx倉儲更新

6.git checkout main 切換至main分支

7.git merge upstream/main 合併遠程main分支

8.git push 將本地倉儲推送至fork409232xx後自己名下之倉儲

方法二:

1.先fork大組倉儲到自己帳號下

2.fork完後到自己帳號下的大組倉儲 Settings⇒pages 建立網頁

3.各組大組倉儲去 Settings⇒Collaborators⇒Manage access⇒add people 加入自己的組員，讓組員有推送權限

4.黑窗(cmd)打指令抓取大組倉儲 git pull --recurse-submodules git@github.com:40923252/cd2022b3.git 紅字部分有關於推送時的權限

5.更改大組網頁內容後 acp "" 更新大組網頁

6.自己帳號下的大組倉儲利用逆向 Pull requests 使自己帳號下的大組倉儲和大組倉儲網頁同步。 Pull requests⇒New pull request 進到此頁面後上方的網址兩個帳號名字互換後直接Enter再按 Create pull request⇒Create pull request⇒Merge pull request⇒Confirm merge
